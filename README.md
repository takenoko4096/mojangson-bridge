# mojangson-bridge

json <-> mojangson <-> nbt の相互変換

## Javadocs

- [0.1.0](https://takenoko4096.github.io/mojangson-bridge/0.1.0)

## `pom.xml`

```xml
<dependency>
    <groupId>io.github.takenoko4096</groupId>
    <artifactId>mojangson-bridge</artifactId>
    <version>0.1.0</version>
</dependency>
```

## `build.gradle.kts`

```kotlin
repositories {
    mavenCentral()
}

dependencies {
    implementation("io.github.takenoko4096:mojangson-bridge:0.1.0")
}
```
